  • Physics for Video Games (PHEN 100)

  • Pixel Coordinate System - This demonstrates the pixel coordinate system used in Processing's display window: the location of the origin and the positive direction of each axis. This also demonstrates the use of Processing's size function for changing the width and height of the display window and Processing's mouseX and mouseY variables for determining the position of the mouse in the display window.
  • Coming soon... World Coordinate System 1
  • World Coordinate System 2 - This demonstrates my world coordinate system: the location of Processing's display window in the world and the positive direction of each axis. This also demonstrates the use of my setDisplayXInWorld, setDisplayYInWorld, setDisplayWidthInWorld and setDisplayHeightInWorld functions for changing the location, width and height of the display window in the world and my mouseXInWorld and mouseYInWorld variables for determining the position of the mouse in the world.
